Simon Game

- The goal is to repeat the sequence played by the computer 20 times.
- If the strict mode is off (strict button red) and the user plays the sequence incorrectly, the computer plays the sequence again without restarting the game.
- If the strict mode is on (strict button green) and the user plays the sequence incorrectly, the game ends. The user restart the game with a new sequence.
